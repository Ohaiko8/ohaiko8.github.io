    Assignment

    Create a web version of countdown timer.

    Introduction

    Developing a timer would be too easy for me, so I decided to make it more difficult. I made a fully functional stopwatch. I wouldn’t say that something complicated is used in the code, rather, it’s just that the stopwatch has the ability to make functionality for different modes of operation. For example, reset, or stop, or record the result, and so on. In this regard, it gives more opportunities to think about how best to do it.

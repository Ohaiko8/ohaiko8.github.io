    • Fancy Timer

    Assignment

    Today we covered css-animations and we explored how we can uses css to create interactive elements like menus etc.

    The challenge for next week is to make a fancy digital count down timer using only pure html+css.

    View

    Code Explanation

    The first picture shows us how I solved the problem that we cannot use js in this task. I used a number of spans in each of which I placed a number, which will then shown on the timer. Also, a little lower, I placed similar spans in order to make the effect of the disappearance of numbers with a delay, as well as numbers in the background, in the future.

    The second picture shows the code that I wrote in order for our numbers to change over time. Here I had to experiment a little with the animation to make it work correctly.

    The last screenshot shows styles for those animations that you can see on the timer. Namely, the smooth disappearance of numbers and numbers in the background. I would say that this was the hardest part in terms of both coming up with the implementation and picking the right values for the parameters.

    Conclusion

    Designing the timer was more interesting than I thought it would be. To begin with, I read the examples the teacher shared with us. I combined some of them and tried to do it. The only problem I had was that I couldn't find any samples online without js, so I had to improvise.

    The result was quite interesting. I wouldn't say that I've noticed something new, because I already have a lot of experience in css and html, but it was interesting to guess how to make the timer without js, although it can not be called full-fledged.

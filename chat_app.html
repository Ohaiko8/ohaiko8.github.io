    Assignment

    Develop a chat app written on Node.js.

    Introduction

    It was the hardest task. In it, I had to use a Node js server as well as Socket io to create a simple chat. The biggest challenge was to correctly connect and configure Socket IO, as it constantly gave errors and did not work at startup. It took me 4 hours to figure out what the problem was. But as a result, everything works. In the future, it would be nice to add different users so that the name is displayed when sending a message.

    I also added as many notes in the code as possible so that in the future I can go back and remember the process without difficulty.

    Code Explanation

    To begin with, we create the main variables that will be useful to us later. The most interesting thing here is the creation of a variable for field. For this we use array. In the variable of field we have 0 - points that the pacman will eat, 1 - wall, 2 - ghost lair, 3 - power pallet, 4 - nothing.

    Next, we create a board and add layout to it.

    Next, one of the most important parts of the code is adding a pacman to the playing field, as well as a function to describe its movement.

    Next comes a function to describe what happens when we eat a dot. In our case, the point disappears, score is added and updated on the page.

    We describe a function to describe what happens when a pacman eats a power pallet. First we add 10 points, change the parameters of ghosts and make them blink. After 10 seconds, we return the casts to normal and remove the power pellet.

    Next is a very simple function to bring the ghosts into normal mode and make them non-blinking.

    We describe the ghost in the class and add a constructor. We also describe it.

    Adding ghosts to the board.

    Great function to describe the movement of ghosts. They move 1 square in a random direction. More explanation of what the code is in the screenshot.

    Next is a function to describe what happens when the game is lost. We remove ghosts, prevent Pac-Man from moving, and also show a notification that the game is lost.

    We make a function to describe what happens when the game is won. If our score is 274, then we remove the ghosts, remove the pacman's ability to move, and show a notification that the player has won.

    And finally, a css file with all the styles for our game.

    Conclusion

    This assignment is the hardest one we've ever been given. It uses a huge amount of js and it was very hard to figure out how to properly make this game.

    To be honest, while watching the class, I didn't notice the example the teacher sent, so I had to think for myself. I killed quite a bit of time trying to do it on my own, but it didn't work out in the end. I decided to go to the tutorials on YouTube and try to do something like that. Anyway, the point is not to sit for 10 hours on the same task, but to gain new knowledge. I watched the two hour course and finally got around to making this game. I learned a huge amount of stuff like addEventListener, removeEventListener, setInterval, Math.random() and others.

    And so when I spent a huge amount of time, I noticed that there was an example code attached to the assignment. But actually I'm even glad I didn't notice it in time, as I gained an incredible amount of experience from this assignment.

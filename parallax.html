    • Parallax

    Assignment

    Create a parallax illustration.

    Introduction

    Here I worked with a very interesting Swiper library. With its help, you can create very beautiful parallax effects. The library itself was not difficult, at least for ordinary tasks, it only took me 1 day to figure it out.

    As a basis, I decided to take images from the game "The Witcher 3" and distribute them into layers in Photoshop. This came in handy in the future to create a 3D effect. I also decided to make the website horizontal just for experience since I didn't know how to do it before. Well, from interesting finds, I found a script for adding Particles animation to the image.

    I also wanted to insert the music from the game, but as it turned out, Chrome does not support autoplay, so I decided to abandon this idea. Asking what was the most difficult part of this task, I can highlight getting to know Swiper and fixing some bugs that occurred with the image during animation.

    Code Explanation

    First, we create an html file and include the rest of the files. Also, do not forget about the additional libraries Swiper and Particles. They are very important to this project.

    Next, we create a simple document structure for Swiper to work with. Here it is necessary to observe the names of the classes for the library to work correctly. Next, we add our photos separated into different layers to create a parallax effect. We also configure this parallax in "data-swiper-parallax". This parameter is responsible for how fast our pictures move in relation to each other. And also don't forget to add particles. Pay attention to exactly where we add particles.

    Next, we create a separate js file to configure the work of Swiper. Everything is pretty simple here. All that is required of us is simply to specify the speed, effect, distance between photos and scrolling with the mouse wheel.

    Conclusion

    In conclusion, I will say that the task was very interesting and informative. It seems to me that with the help of such parallax effects it will be possible to create a wow effect in the future when creating websites. Of course, it won't look good on every website, but I think that gaming websites, or some kind of art-related websites, would be very suitable for this feature.

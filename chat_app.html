    Assignment

    Develop a chat app written on Node.js.

    Introduction

    It was the hardest task. In it, I had to use a Node js server as well as Socket io to create a simple chat. The biggest challenge was to correctly connect and configure Socket IO, as it constantly gave errors and did not work at startup. It took me 4 hours to figure out what the problem was. But as a result, everything works. In the future, it would be nice to add different users so that the name is displayed when sending a message.

    I also added as many notes in the code as possible so that in the future I can go back and remember the process without difficulty.

    Code Explanation

    At the very beginning, I created a package.json file and described my application. Also there will be dependences for connecting socket.io and express.

    I included bootstrap in the html file to make it much easier to work with styles later on.

    Next, I added a navbar which does not carry any function in itself. This was done just for looks. And of course, I just wanted to make a navbar using bootstrap, since I had no such experience before.

    Next, we write the code for the main part on the page. This is the chat itself.

    Next, we write js code in order to make our chat work. Initially, we include jQuery and socket. These are the 2 main components on which all our code rests. After that, we turn on the socket and check the connection. Next, we describe what happens when you click on the send message button. We send a message to the socket and clear the field to enter the message. Next, we send a message to all chat users and add this message to the page.

    We create js file "index.js" and add variables to configure how the server works. They can be taken from the socket io official website.

    Then we get the url address and display the page.

    Setting up the port for our server.

    We create an array in which all connections on the server will be stored.

    Adding an event to connect and disconnect from the socket server.

    Well, at the end, the code for receiving and displaying the message.

    In order to start the server, you need to write "node index.js" in a terminal and enter "https://localhost:3000/".

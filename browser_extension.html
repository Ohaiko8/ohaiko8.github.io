    Assignment

    Develope a chat app written on Node.js.

    Introduction

    When I saw this task in the Intermediate tab, I thought it was some kind of mistake, because I thought that developing browser extensions is much more difficult. But as it turned out, the difficulty is not in porting your application in the browser as an extension, but in writing functionality. But it ended up being a lot easier than I thought. I made a rather banal extension that allows you to download a screenshot of an entire page, or its individual element.

    Conclusion

    In conclusion, I will say that this task was really very interesting for me, because now I know that making browser extensions is not so difficult. I learned a lot, but I especially want to pay attention to using a manifest file with settings (I have not had to do this before) and working with keystrokes (I did something like this when I worked on Unity, but this was the first time in js).
